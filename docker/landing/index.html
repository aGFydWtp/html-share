社内向けの HTML 共有

作った HTML を、@<WORKSPACE_DOMAIN> でログインした社内メンバーだけが見られる URL で配ります。社外からは到達できません。

HTML をアップロード 自分の共有を管理

Web でアップロード

/upload に単一 HTML をドラッグ＆ドロップすると、共有 URL がその場で発行されます（25MB まで）。

  • 共有 URL は IAP 認証済みの社内限定。社外には公開されません。
  • 作成は誰でも可。上書きはできず、毎回あたらしい URL が発行されます。
  • 自分が上げたものは /mypage から削除できます。
詳しい使い方は リポジトリの README を参照してください。
