html-share

作った HTML を、社内限定の URL でさっと共有するための基盤です。
閲覧も共有も @prc-dev.persol-group.co.jp の Google アカウントでログインした社内メンバーだけ。社外からは到達できません。

HTML をアップロードする → 自分の共有を管理 →

Web でアップロード

/upload に単一 HTML をドラッグ＆ドロップ。共有 URL がその場で発行されます（25MB まで）。

  • 発行された URL は社内（IAP 認証済み）限定。社外には公開されません。
  • アップロードは誰でも可。上書き・削除は不可で、毎回あたらしい URL が発行されます。
  • 削除が必要なときは管理者の操作で行います。
社内向け · 詳しい使い方は リポジトリの README を参照してください。
